# FocusFlow

> **Note:** This is a fictional SaaS project created as a portfolio project to showcase my frontend development skills.

## Description

FocusFlow is an AI-powered productivity tool designed to help you organize your day, automatically generate tasks, and boost your productivity. Powered by artificial intelligence, FocusFlow learns your habits and priorities to create a personalized workflow experience.

## Features

- 📋 **Smart Task Generation** - AI automatically creates tasks based on your goals
- 🎯 **Day Organization** - Organize and prioritize your daily activities
- 🤖 **AI-Powered** - Machine learning algorithms optimize your workflow
- ⚡ **Boost Productivity** - Improve focus and task completion rates

## Tech Stack

- [Astro](https://astro.build) - Modern web framework
- [Tailwind CSS](https://tailwindcss.com) - Utility-first CSS framework
- [pnpm](https://pnpm.io) - Fast, disk space efficient package manager
- **Dark/Light Mode** support

## Getting Started

### Prerequisites

- Node.js 16+
- pnpm

### pnpm Commands Pack

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build

# Preview production build
pnpm run preview

# Upgrade dependencies
pnpm upgrade

# Update pnpm
pnpm update
```
